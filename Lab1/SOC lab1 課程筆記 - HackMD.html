Multip.py程式解釋[2]
































# coding: utf-8

# In[ ]:


from __future__ import print_function

import sys, os

sys.path.append('/home/xilinx')
os.environ['XILINX_XRT'] = '/usr'
from pynq import Overlay

if __name__ == "__main__":
    print("Entry:", sys.argv[0])
    print("System argument(s):", len(sys.argv))

    print("Start of \"" + sys.argv[0] + "\"")

    ol = Overlay("/home/xilinx/jupyter_notebooks/Multip2Num.bit")
    regIP = ol.multip_2num_0
    
    for i in range(9):
        print("============================")
        for j in range(9):
            regIP.write(0x10, i + 1)
            regIP.write(0x18, j + 1)
            Res = regIP.read(0x20)
            print(str(i + 1) + " * " + str(j + 1) + " = " + str(Res))
    print("============================")
    print("Exit process")
  1. from future import print_function：導入一個與Python 2.x和Python 3.x兼容性相關的模組，確保在Python 2.x環境中也能使用Python 3.x的print函數。
  2. sys.path.append(‘/home/xilinx’)：將’/home/xilinx’路徑添加到Python的模組搜索路徑中，以便可以導入該目錄下的模組。
  3. os.environ[‘XILINX_XRT’] = ‘/usr’：設置名為’XILINX_XRT’的環境變數為’/usr’，可能是與Xilinx工具鏈相關的配置。
  4. from pynq import Overlay：從pynq模組導入Overlay類別，這個類別用於載入FPGA的Overlay文件。
  5. if name == “main”:：這是Python中的主程式入口點，表示以下程式碼將在腳本被直接執行時運行。
  6. ol = Overlay(“/home/xilinx/jupyter_notebooks/Multip2Num.bit”)：使用指定的Overlay位文件創建Overlay物件。
  7. regIP = ol.multip_2num_0：從Overlay物件中獲取名為multip_2num_0的IP核（硬體模組）的寄存器介面。
  8. regIP.write(0x10, i + 1)：向IP核的地址0x10寫入當前行數加1的值。
  9. regIP.write(0x18, j + 1)：向IP核的地址0x18寫入當前列數加1的值。
  10. Res = regIP.read(0x20)：從IP核的地址0x20讀取計算結果。

這段程式碼的主要功能是通過FPGA硬體實現乘法計算，並輸出乘法表格中的乘積結果。硬體加速部分可能由Overlay文件定義，其中的multip_2num_0是與FPGA上的IP核（硬體模組）關聯的名稱。在迴圈中，不斷向硬體模組寫入輸入值並讀取輸出值，以便進行乘法計算。
